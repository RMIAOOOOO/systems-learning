# Daily LeetCode

Solve **one LeetCode problem every weekday** following the NeetCode roadmap.

Guidelines:

* Prioritize understanding over memorization.
* Review time and space complexity.
* Refactor the solution into clean, modern C++.
* Record key takeaways in your notes.

**Goal:** Build problem-solving skills and reinforce modern C++ through consistent practice.
